--
- ข้อมูลการเชื่อมต่อฐานข้อมูล Private Key จะแจ้งให้ทราบในห้องสอบ
- โครงสร้าง JSON สำหรับจัดเก็บข้อมูลภายใต้คอลเลกชัน (Collections) Herbs
{
    hbId: String,
    hbName: String,
    hbDesc: String,
    hbCate: Number,
    hbProp: String,
    hbSupp: String
}

กำหนดให้เพิ่มเติมการทำงานของแอปพลิเคชันให้ทำงานได้อย่างสมบูรณ์ ดังนี้
- เชื่อมโยงหน้าเพจในทุกหน้าให้สมบูรณ์
- กรอกข้อมูลในฟอร์มเพื่อดำเนินการบันทึก โดยระบุข้อมูลในหนึ่งรายการที่เพิ่มชื่อ-สกุลนักศึกษาในฟิลด์ผู้ผลิต เช่น Anna Devid xxxxxxxxxxxxxx-x ในคอลเลกชัน Herbs อย่างน้อย 2 รายการ
- กรอกข้อมูลแก้ไขข้อมูลสมุนไพร โดยระบุข้อมูลในหนึ่งรายการที่เพิ่มลงท้ายรหัสนักศึกษาในฟิลด์ชื่อสมุนไพร เช่น HerbA xxxxxxxxxxxxxx-x ในคอลเลกชัน Herbs เพียง 1 รายการ
- กำหนดให้ดึงข้อมูลสมุนไพรในคอลเลกชัน Herbs แสดงผลทุกเอกสาร (Documents)
- กำหนดให้เลือกลบรายการสมุนไพรในคอลเลกชัน Herbs ที่ไม่ต้องการได้
- กำหนดให้ดึงรายการสมุนไพรในคอลเลกชัน Herbs แสดงผลในแต่ละเอกสารเพื่อดูรายละเอียดได้

การเริ่มต้นทำงานแอปพลิเคชันฝั่ง Client
> npm run dev

การเริ่มต้นทำงานแอปพลิเคชันฝั่ง Server
> npm run server
